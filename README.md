# Bayesian Optimization for Hyperparameter Tuning

This repository contains the code and tutorial materials for an MSc Data Science individual assignment on **Bayesian Optimization for Hyperparameter Tuning**.

## Repository contents

- `notebooks/bayesian_optimization_tutorial.ipynb` — full Jupyter notebook used to generate the experiments and plots.
- `tutorial_source.md` — tutorial text that can be exported to PDF.
- `requirements.txt` — Python dependencies.
- `LICENSE` — MIT licence.

## How to run

1. Create a virtual environment.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Launch Jupyter:

```bash
jupyter notebook
```

4. Open `notebooks/bayesian_optimization_tutorial.ipynb` and run all cells.
